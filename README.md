# بطاقات تهنئة عيد الأضحى المبارك

موقع بسيط لإنشاء بطاقات تهنئة عيد الأضحى بتصميم عربي إسلامي أصيل، يعمل بالكامل في المتصفح — بدون تسجيل دخول وبدون قاعدة بيانات.

## المميزات

- واجهة عربية كاملة بدعم RTL.
- إدخال الاسم ورسالة تهنئة اختيارية.
- رفع صورة شخصية أو شعار (PNG / JPG) ومعاينة فورية.
- 4 قوالب فاخرة كلها لعيد الأضحى مع أيقونة الخروف وزخارف إسلامية:
  1. الذهبي الكلاسيكي
  2. الزمردي الأصيل
  3. الملكي الفاخر
  4. الصحراوي العصري
- معاينة مباشرة للبطاقة أثناء الكتابة.
- تنزيل البطاقة كصورة PNG عالية الجودة (دقة 2x).

## التشغيل

لا حاجة لأي بناء أو خادم — فقط افتح `index.html` في المتصفح:

```bash
open index.html
``````

## البنية

```
.
├── index.html      # الصفحة الرئيسية
├── styles.css      # التصميم العربي الإسلامي
├── app.js          # منطق التطبيق والمعاينة والتنزيل

```

## التقنيات

- HTML / CSS / JavaScript بدون أي إطار عمل.
- خطوط Google: Aref Ruqaa, Amiri, Cairo, Reem Kufi, Lateef.
- [html-to-image](https://github.com/bubkoo/html-to-image) لتصدير البطاقة كصورة بدقة عالية مع دعم كامل للنصوص العربية.

## الخصوصية

كل المعالجة تتم في متصفح المستخدم؛ لا يتم رفع أي بيانات أو صور إلى أي خادم.
